Naturaleza del Sistema

SICMEC es una solución Web Application integral bajo el entorno Laravel/PHP.


Diseñada para sistematizar procesos de captura de datos de beneficiarios, gestión de inventario crítico y trazabilidad de insumos biomédicos.

Innovación Social

Automatización de procesos administrativos que tradicionalmente dependen de carga manual y colapsos logísticos.

Características 5.0

Chatbot Telegram

Webhook HTTPS oficial. Consultas en tiempo real (0.2s) para estatus de trámites desde el hogar.

SMS Omnicanal

Integración Twilio API. Notificaciones nativas GSM para ciudadanos sin conexión a datos móviles.

PDF Dinámicos

Motor DomPDF para generación de actas de entrega e informes bajo estándares ISO.

Arquitectura Técnica

Framework: Laravel 8.75
Engine: PHP 8.1 (OOP)
Database: MariaDB / MySQL
Frontend: Bootstrap 5 + Vue.js
Notif: Twilio + Telegram API

Seguridad y Rapidez

  • Encriptación Base64
  • Zero-Downtime Config
  • Chart.js Real-time
